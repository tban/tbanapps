# Reglas del Proyecto (TbanApps Showcase)

- **Publicación Automática**: Cada vez que se realicen modificaciones en los archivos del sitio web (`index.html`, `styles.css`, `app.js`, `data.js`, etc.), el agente debe compilar, confirmar (`git commit`) y subir los cambios (`git push`) automáticamente al repositorio de GitHub para publicarlos de inmediato en GitHub Pages, sin necesidad de confirmación o solicitud explícita por parte del usuario.
